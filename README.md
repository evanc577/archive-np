# Download Naver Post images

![Terminal recording](term.svg)


```
USAGE:
    download-np [OPTIONS]

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -d, --directory <DIR>       Directory to download to (default: ./posts)
    -f, --filter <REGEX>        Regex filter on NP title
    -l, --limit <POSTS>         Limit number of posts to process
    -m, --member <MEMBER_ID>    Set NP member id (default: 29156514)
    -u, --url <URL>...          Download NP URL
```
